# fdf
This project is about creating a simplified 3D graphic representation of a relief landscape.

Give a file as parameter. For example - ./fdf map

## Control:
- ### Movement
   + #### left right up down
- ### Rotation
   + #### NUM8 NUM2 - x
   + #### NUM4 NUM6 - y
   + #### NUM7 NUM9 - z
- ### Change projection (parallel or central)
   + #### space
